如何评价SN打野SOFM在《英雄联盟》2020年lpl全明星周末获得年度mvp奖？ 小妖： 你问我离谱吗。我会说，离谱。这是最离谱的？我回答：不是。我见过的s7才是最离谱的。说实话，仿照nba，足球完善荣誉体系是好事。但藤井这么多年造的荣誉为什么立不住？为什么观众还是只认冠军？原因就在于此了。其实没必要看的太重。记住，观众眼里，赞助商眼里，冠军才是唯一，外战漂亮=吸粉。藤井的荣誉=cjb。至于cjb是什么。问问s8之前的uzi，肉鸡粉丝，问问s10之前的k9粉丝，他们对这个词熟。至于sn。看看就得了。这企业在哪都让人呕吐。只会营销的货色。张康阳当初世界赛一顿营销，精英，年少有为人设玩的贼响，各种角度踩着wsc上位。结果国米欧联被干碎，苏宁小店赔的比wsc还多。差不多得了。张康阳那句在意大利提到苏宁，老百姓眼里满满的敬畏。都被人嘲出花了。现在又在lpl搅风搅雨的。
